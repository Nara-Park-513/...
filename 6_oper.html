논리연산자

논리연산자는 변수나 값 사이에 논리를 결정하는데 사용됩니다

&& AND => (x < 10 && y > 1) 두가지 요건이 성립해야만 참입니다
|| or 둘중에 하나만 성립하면 참
! NOT 같지 않으면 true

비교연산자 : 두값을 비교하는데 사용합니다

== : 타입을 고려하지 않은 같은지를 확인
5[숫자] == '5'[문자열] 타입을 고려하지 않은 같음 true
=== : 타입까지 고려하여 같음 5===5 5==='5'
!= : 같지 않음을 확인 같지 않으면 진실 같으면 거짓
!== : 값도 틀리고 타입도 틀린 엄격하게 같지 않음
> : 보다 큰
< : 보다 작은
>= : 크거나 같은 것 중 둘중에 하나만 맞으면 성립
<= : 작거나 같은 것 중 둘중에 하나만 맞으면 성립

할당연산자

변수에 값을 할당합니다

= : 값을 할당
+= : 기존의 값에 값을 추가 add
-= : 기존의 값에서 값을 뺴주는 것
*= : 기존의 값을 곱해주는 것
/= : 나누기는
%= : 나눈 나머지는
**= : 거듭제곱
